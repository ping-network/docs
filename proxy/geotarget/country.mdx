title: "Country"
---

You can target a specific country by using the `country` parameter in your proxy string.  
This parameter allows you to connect to a proxy located in a specific country.  

You can specify any country in the ISO 3166-1 alpha-2 format (e.g., `US` for the United States, `CA` for Canada).  

### Code Example

Example: request from Turkey (cURL):

```bash
curl -x tr-pr.proxy.pingnetwork.io:30000 -U "your-username:your-password" https://ipinfo.io/json
