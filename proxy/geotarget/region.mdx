You can target a specific region (state) by specifying the `region` in the proxy string. Regions can be things like states, provinces, or territories depending on the country. Valid examples are `california`or `southern.cape`.

### Valid examples

st-us_california-city-los_angeles\
st-us_illinois-city-springfield\
st-us_michigan-city-detroit\
st-us_ohio-city-columbus\
st-us_maine-city-portland\
st-us_tennessee-city-memphis

## Code example

Example: request from **California, USA** (cURL):

```bash
curl -x proxy.pingnetwork.io:7776 -U "your-username-st-us_california:your-password" https://ipinfo.io/json
```
